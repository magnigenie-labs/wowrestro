WOWRestro is an online ordering system for WooCommerce which makes it easier to receive takeaway and delivery orders.

== Description ==

WOWRestro is an online ordering system powered by WooCommerce. WOWRestro Allows you to offer both Takeaway/Pickup and Delivery service to your customers. Using WOWRestro you can easily create a responsive food ordering system for free.

Now you can easily convert your website visitors to customers using WOWRestro ordering interface. WOWRestro helps you to convert your WooCommerce store to an online takeway/delivery system by providing a modern and responsive menu design.

Some of the possible use cases of WOWRestro include Online food delivery/takeaway business, Online Grocery delivery/takeaway business, Online flower delivery/takeaway system, Laundry service etc.

WOWRestro enables you to easily add modifiers/options/add-ons for your food items directly from the product add/edit screen of WooCommerce.

The layout of WOWRestro gets implemented on your site using [WOWRestro] shortcode. The plugin automatically creates a page called ORDER ONLINE which basically contains the WOWRestro shortcode and is used for the ordering interface that is presented to your customers.

= Shortcode Attributes =

* category: Comma separated list of category slugs.
* show_search(yes,no): Allows you to show/hide search bar.

= Shortcode examples =

* [wowrestro]
* [wowrestro category="main-course,lunch,starters"]
* [wowrestro category="main-course,lunch,starters" show_search="no"]

= Why use WOWRestro =

* WOWRestro is built with mobile first design approach as most of your customers would use their mobile phones to order online.

* Powered by WooCommerce, which makes it more powerful.

* Ability to easily add modifiers for your items.

* Allows you to set items with Veg or Non-Veg.

* Support for variable products.

* Completely developer friendly with sufficient hooks to alter any functionality of the plugin.

== Installation ==

1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for WOWRestro
3. Click Install and activate the plugin.
4. Once the plugin is activated it will create a page called Order Online with shortcode [wowrestro]. This page will list all your items.
5. Once you have activated the plugin you can navigate to WOWRestro menu on your dashboard and start setting up different settings for the plugin.

== Frequently Asked Questions ==

= How can I add my products to the order online page? =
You can easily add your products from the WooCommerce Products menu. You just need to make sure the product you are adding is marked as WOWRestro item.

= Does it requires WooCommerce? =
Yes, the plugin is built as an extension to WooCommerce so it's really necessary to have WooCommerce installed and active before adding this plugin.

= Do I need any specific theme to make this plugin work correctly? =
No, you don't need any special theme to run this plugin. It works with all themes that support WooCommerce.
